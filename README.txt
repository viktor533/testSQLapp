Тестовое задание для работы с MySQL таблицей. Реадизовано с использованием Swing.

Таблица состоит из столбцов: Имя, Фамилия, Отчество, Дата рождения и Пол.
Есть режим автоматической генерации таблицы, в котором генерируется 1000 произвольных элементов таблицы,
в 100 из которых фамилия начинается на заданную букву.
Есть возможность вывода всех элементов таблицы, всех уникальных, а так же
всех, в которых фамилия начинается на заданную букву.
Так же имеется возможность ручного добавления элементов в таблицу и их просмотра.
Все настройки соединения задаются в окне настроек.
